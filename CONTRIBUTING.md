# Contributing to IMLADRIS

Welcome, Wanderer. Thank you for your interest in helping us expand the Knowledge Sanctuary.

## Principles of Contribution
Imladris is more than a tool; it is a design-first experience. All contributions must adhere to our core principles:
- **Aesthetic Excellence**: Every UI change must feel ethereal, calm, and majestic.
- **Privacy First**: No features requiring external cloud services will be accepted.
- **Clean Architecture**: Maintain the separation between Data, Domain, and Presentation layers.

## Development Workflow
1. Fork the repository.
2. Create a feature branch (`git checkout -b feature/sanctuary-expansion`).
3. Commit your changes with meaningful, emoji-free messages.
4. Push to your fork and submit a Pull Request.

## Design Guidelines
If you are contributing UI elements:
- Use **Glassmorphism** (semi-transparent backgrounds with blur).
- Favor #E1E8ED (Silver) for text and #D4AF37 (Gold) for accents.
- All animations should use `SineEaseInOut` or similar gradual curves.
