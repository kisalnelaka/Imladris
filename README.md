# IMLADRIS

**IMLADRIS** is a next-generation offline-first Android library app designed as a living knowledge sanctuary. Inspired by the elegance of Rivendell, it transforms your digital bookshelf into a calm, ethereal mind palace.

![Imladris Feature Graphic](docs/assets/feature_graphic.png)

## Core Vision
Imladris is designed to be more than a file manager. It is an intelligent space built for:
- **Calm Perception**: Glassmorphic UI with soft glowing lights.
- **Deep Wisdom**: Knowledge graph visualization of your thoughts and books.
- **Privacy First**: Fully offline, utilizing Android System Access Framework (SAF).

## Key Features
- **Spatial UI**: Navigate through 'Gateways' - glowing portals to your collections.
- **Knowledge Graph**: Visualize thematic connections between different artifacts.
- **Ethereal Reader**: A focus-optimized reading experience with elegant serif typography.
- **Reading Analytics**: Track focus scores and your intellectual timeline.
- **Memory Recall**: Gentle, non-intrusive notifications that resurface past highlights.
- **Smart Widgets**: Resume your journey directly from your home screen.

## Technology Stack
- **Language**: Kotlin 1.9+
- **UI**: Jetpack Compose & Jetpack Glance
- **Architecture**: Clean Architecture (Layered)
- **Dependency Injection**: Hilt
- **Persistence**: Room Database
- **Background Work**: WorkManager

## Getting Started
### Prerequisites
- Android Studio Iguana+
- JDK 17
- Android SDK 34 (Upside Down Cake)

### Build Instructions
1. Clone this repository.
2. Open the project in **Android Studio**.
3. Let Gradle sync and download dependencies.
4. Click **Run** or use `.\gradlew assembleDebug` (after sync).
